# Customer Data Management & Automation System

A complete Python application to manage customer records using MySQL, with automatic reporting and data processing

---

## Features

- Add, View, Update, and Delete customer records
- Automatic calculation of total spending and customer statistics
- Identify VIP customers (spending ≥ RM500)
- Generate summary reports in **TXT format**
- Export full customer data in **CSV format** (compatible with Excel)
- Input validation and error handling
- Data integrity using commit and rollback

---

## Technologies Used

- **Python 3**
- **MySQL / phpMyAdmin**
- **Library:** `mysql-connector-python`

---

## How to Run

### 1. Install required library

Open your terminal and run:

```bash
pip install mysql-connector-python
```
